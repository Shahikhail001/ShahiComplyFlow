# Exporting User Data for DSR Requests

## Overview
Export all user data in response to Data Subject Rights (DSR) requests for compliance with privacy laws.

## Steps
1. Go to **WordPress Admin → ComplyFlow → DSR Portal**.
2. Find the relevant DSR request and click to view details.
3. Click **Export Data** and choose the desired format (JSON, CSV, XML).
4. Download the exported file and provide it to the user as required.

## Tips
- Always verify the request before exporting sensitive data.
- Use the audit trail to log all exports for compliance.
- Exported data includes user info, orders, comments, and more if integrated.
