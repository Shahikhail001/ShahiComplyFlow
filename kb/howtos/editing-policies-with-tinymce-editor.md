# Editing Policies with TinyMCE Editor

## Overview
Edit legal documents using the built-in TinyMCE rich text editor for full control over formatting and content.

## Steps
1. Go to **WordPress Admin → ComplyFlow → Legal Documents**.
2. Click **Edit** on the policy you want to modify.
3. Use the TinyMCE editor to make changes:
   - Switch between Visual, HTML, and Preview tabs.
   - Use formatting tools (bold, italic, lists, links, etc.).
   - Edit HTML directly for advanced customization.
4. Click **Save Changes** to update the policy.

## Tips
- Use the Preview tab to see how the policy will appear to users.
- Use Fullscreen mode for distraction-free editing.
- All changes are versioned for easy rollback.
