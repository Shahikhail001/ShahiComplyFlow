# Managing Vendor Relationships

## Overview
Track and manage third-party vendors and their compliance status using the Vendor Management module.

## Steps
1. Go to **WordPress Admin → ComplyFlow → Vendor Management**.
2. Add new vendors manually or let the system auto-detect scripts and integrations.
3. Upload Data Processing Agreements (DPAs) and set renewal reminders.
4. Assign risk scores and compliance status to each vendor.
5. Monitor compliance changes and receive alerts for expiring agreements.

## Tips
- Keep vendor information up to date for audits.
- Use risk assessment to prioritize vendor reviews.
- Regularly review compliance status and update DPAs as needed.
