# Processing DSR Requests

## Overview
Handle and respond to Data Subject Rights (DSR) requests from users efficiently and in compliance with regulations.

## Steps
1. Go to **WordPress Admin → ComplyFlow → DSR Portal**.
2. Review the list of pending, verified, in-progress, and completed requests.
3. Click on a request to view details and user-submitted information.
4. Use the available actions to verify, process, complete, or reject the request.
5. Add notes or comments for audit purposes.
6. Export request data as needed (JSON, CSV, XML).

## Tips
- Respond to requests within the required SLA (30/45 days).
- Use the audit trail to track all actions taken.
- Set up email notifications for new and completed requests.
