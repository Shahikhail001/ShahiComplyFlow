# Prerequisites and Requirements

## System Requirements

To ensure ComplyFlow runs smoothly, your environment must meet the following requirements:

- **WordPress Version**: 6.4 or higher
- **PHP Version**: 8.0 or higher
- **Node.js Version**: 18.0 or higher (for development purposes)
- **Composer**: Installed for managing PHP dependencies

## Recommended Setup

- **Local Development Tools**:
  - Local by Flywheel
  - Docker with wp-env
- **Database**: MySQL 5.7+ or MariaDB 10.3+
- **Web Server**: Apache or Nginx

## Additional Tools

- **Code Quality Tools**:
  - PHPCS (PHP CodeSniffer)
  - PHPStan (Static Analysis)
  - ESLint (JavaScript Linting)
- **Build Tools**:
  - Vite for asset bundling
  - Tailwind CSS for styling

Ensure all tools are installed and configured before proceeding to the installation steps.
