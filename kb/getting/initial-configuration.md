# Initial Configuration

## Accessing the Settings

1. Navigate to the WordPress admin dashboard.
2. Click on **ComplyFlow** in the sidebar menu.
3. Open the **Settings** page to configure the plugin.

## General Settings

- **Enable/Disable Plugin**: Toggle the plugin functionality.
- **Data Retention Period**: Set the duration for retaining compliance data.
- **IP Anonymization**: Enable to anonymize user IP addresses for GDPR compliance.

## Module Settings

### Accessibility Scanner
- **WCAG Level**: Choose between A, AA, or AAA compliance levels.
- **Scan Frequency**: Set automated scan intervals (daily, weekly, etc.).
- **Notification Recipients**: Add email addresses for scan reports.

### Consent Manager
- **Banner Position**: Select top, bottom, or modal display.
- **Default Consent State**: Configure opt-in or opt-out behavior.
- **Cookie Categories**: Define categories like Necessary, Analytics, Marketing.

### DSR Portal
- **SLA Days**: Set the response time for data subject requests.
- **Verification Method**: Choose email or manual verification.
- **Anonymization Rules**: Define rules for anonymizing user data.

## Saving Changes

Click the **Save Settings** button after making changes. Ensure the settings are applied correctly by testing the plugin features.
