# PHP Version Compatibility Errors

## Problem
The plugin fails to activate or throws errors due to an unsupported PHP version.

## Solution
- Ensure your server is running **PHP 8.0 or higher**.
- Check your PHP version in WordPress Admin → Tools → Site Health → Info → Server.
- If using shared hosting, update PHP via your hosting control panel or contact your provider.
- After upgrading PHP, restart your web server and try activating the plugin again.

## Tips
- Always back up your site before upgrading PHP.
- Test plugin functionality after any PHP upgrade.
