# Consent Banner Not Displaying

## Problem
The consent banner does not appear on your website.

## Solution
- Ensure the banner is enabled in ComplyFlow settings.
- Clear browser and site cache (including caching plugins/CDN).
- Check for JavaScript errors or conflicts in the browser console.
- Verify that the banner is not hidden by custom CSS or theme overrides.
- Disable other plugins to rule out conflicts.

## Tips
- Use the preview feature in settings to test banner appearance.
- Test on multiple browsers and devices.
