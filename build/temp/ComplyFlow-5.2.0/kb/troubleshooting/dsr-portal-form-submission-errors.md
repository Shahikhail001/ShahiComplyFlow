# DSR Portal Form Submission Errors

## Problem
Users are unable to submit Data Subject Rights (DSR) requests via the portal.

## Solution
- Ensure the DSR portal shortcode is correctly placed on a published page.
- Check for JavaScript or PHP errors in the browser console and debug log.
- Verify email settings for notifications and confirmations.
- Test the form with different browsers and user roles.

## Tips
- Use the audit trail to track failed submissions.
- Regularly test the DSR portal after updates.
