# Running Accessibility Scans Manually

## Overview
Learn how to manually run accessibility scans on your website to check for WCAG 2.2 compliance.

## Steps
1. Go to **WordPress Admin → ComplyFlow → Accessibility**.
2. Select the page, post, or enter a custom URL you want to scan.
3. Click **Scan Now**.
4. Wait for the scan to complete. Results will appear in the scan results table.
5. Review issues by severity and follow remediation guidance.
6. Optionally, export results as CSV for sharing or further analysis.

## Tips
- Run scans after major content or design changes.
- Use scheduled scans for ongoing monitoring.
- For advanced users, use the REST API or WP-CLI to automate scans.
