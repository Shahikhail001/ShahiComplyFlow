# Importing/Exporting Plugin Settings

## Overview
Easily migrate or back up your ComplyFlow settings using the import/export feature.

## Steps
1. Go to **WordPress Admin → ComplyFlow → Settings**.
2. Click the **Export Settings** button to download your current configuration as a JSON file.
3. To import, click **Import Settings** and upload a previously exported JSON file.
4. Review and confirm the import to apply settings.

## Tips
- Always export settings before making major changes or updates.
- Use import/export to quickly configure multiple sites.
- Review imported settings for compatibility with your environment.
