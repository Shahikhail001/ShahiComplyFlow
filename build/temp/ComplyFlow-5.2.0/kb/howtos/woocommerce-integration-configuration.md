# WooCommerce Integration Configuration

## Overview
Enable and configure WooCommerce integration for full compliance coverage of your e-commerce store.

## Steps
1. Ensure WooCommerce is installed and active on your site.
2. Go to **WordPress Admin → ComplyFlow → Settings → Integrations**.
3. Enable WooCommerce integration.
4. Review detected WooCommerce data types (orders, customers, reviews, checkout consent).
5. Configure consent checkboxes and privacy notices for checkout forms.
6. Ensure DSR exports include WooCommerce data.

## Tips
- Regularly review WooCommerce-related compliance settings after plugin/theme updates.
- Test checkout and account pages for consent and privacy notices.
- Use DSR exports to provide users with their order and account data.
