
# Analytics and Reporting

## What is Analytics and Reporting?
The Analytics and Reporting module tracks, visualizes, and exports all key compliance metrics for your website. It helps you monitor progress, identify issues, and generate reports for audits, management, or regulatory requests.

## Key Features

### 1. Compliance Scoring
- **Weighted Algorithm**: Calculates a compliance score (0-100) based on accessibility, consent, DSR, cookie, and legal document status.
- **Deduction Model**: Points are deducted for unresolved issues, missing consents, overdue DSRs, and policy gaps.
- **Score Breakdown**: View detailed scores for each compliance area.
- **Letter Grades**: Get an A-F grade for quick assessment.

### 2. Audit Trail
- **Event Logging**: Every scan, consent, DSR request, and document update is logged with details.
- **Timestamps**: Each event is time-stamped for traceability.
- **Admin Viewer**: Filter, search, and review logs in the WordPress dashboard.
- **Export Logs**: Download logs as CSV for external review or compliance checks.

### 3. Report Generation
- **CSV Export**: Export compliance scores, audit trails, and trend data for analysis or sharing.
- **PDF Export**: Create print-ready reports for management or regulatory bodies.
- **Scheduled Reports**: Set up automated report generation and email delivery.
- **Custom Reports**: Filter by date range, module, or event type.

### 4. Trend Analysis
- **Historical Trends**: Visualize compliance scores and issue counts over time.
- **Module-Specific Trends**: Drill down into trends for accessibility, consent, DSR, and more.

## Troubleshooting & Tips
- If scores seem low, review the detailed breakdown to identify problem areas.
- For missing events, ensure all modules are enabled and logging is active.
- Use the audit trail to resolve disputes or demonstrate compliance during audits.
- Schedule regular exports for backup and compliance documentation.

## Advanced Usage
- **REST API**: Access analytics and logs programmatically for integration with BI tools.
- **WP-CLI**: Use CLI commands to export or analyze data in bulk.
- **Custom Dashboards**: Developers can build custom dashboards using the REST API and hooks.
